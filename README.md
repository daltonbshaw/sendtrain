# Send Train 🧗

A periodized 8-week climbing training plan web app, built for advanced climbers (V9+) peaking in August.

## Features

- **3-phase periodization** — Base Strength (Wks 1–3) → Deload (Wk 4) → Power (Wks 5–6) → Performance Peak (Wks 7–8)
- **Weekly calendar** — click any day for full session details
- **Structured gym sessions** — every Tuesday/Thursday shows Warm-Up, Climbing, and Strength & Flexibility sections, adapted per phase
- **8-week roadmap** — jump to any week at a glance
- **Session tracker** — mark sessions done or skipped, saved to localStorage

## Deploy with GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, root folder
4. Your app will be live at `https://<your-username>.github.io/<repo-name>`

## Local use

Just open `index.html` in any browser — no server, no dependencies, no build step required.

## Stack

Pure HTML, CSS, and vanilla JavaScript. No frameworks, no bundler. Single file.
